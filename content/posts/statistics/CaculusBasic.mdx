1) 연속성과 미분가능성의 반례들
- 단순히 그래프가 이어져 있다고 생각하는 수준을 넘어야 함.
- Weierstrass Function: 모든 점에서 연속이지만 모든 점에서 미분 불가능한 함수의 존재를 머릿속에 두어야 함.
    - 함수 $f(x) = x^n \sin(1/x)$ (at $x=0$): * $n=1$일 때: $x=0$에서 연속이지만 미분 불가능.
        - $n=2$일 때: $x=0$에서 미분 가능하지만, 도함수 $f'(x)$는 $x=0$에서 불연속

2) 평균값 정리 (Mean Value Theorem, MVT)의 응용
- 단순히 "기울기가 같은 점이 존재한다"를 넘어, 부등식 증명에 활용해야 함.
- 예시 문제 유형: $f(0)=1$이고 모든 $x$에 대해 $|f'(x)| \leq 2$일 때, $f(3)$의 가능한 범위는?
    - 풀이: MVT에 의해 $\frac{f(3)-f(0)}{3-0} = f'(c)$인 $c$가 존재합니다. 따라서 $f(3) = f(0) + 3f'(c)$입니다. $|f'(c)| \leq 2$이므로 $-6 \leq 3f'(c) \leq 6$이 되어, $1-6 \leq f(3) \leq 1+6$, 즉 $-5 \leq f(3) \leq 7$ 임.

3) 이상 적분 (Improper Integrals)의 수렴 판정
- 통계학에서 확률 밀도 함수를 다룰 때 매우 중요한 기초
    - $`\int_{1}^{\infty} \frac{1}{x^p}dx`$ : p > 1일 때 수렴.
    - $`\int_{0}^{1} \frac{1}{x^p}dx`$ : p < 1 일 때 수렴.
    - Comparison Test: 복잡한 함수가 주어지면 더 단순한 $`\frac{1}{x^p}`$ 형태와 비교하여 수렴 여부를 빠르게 판단하는 연습이 필요
